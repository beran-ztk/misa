# MISA

MISA is an orchestration system for work, knowledge, and long-term decision making.

Instead of using separate tools for tasks, notes, planning, and knowledge management, MISA models everything as structured items inside one system. Tasks, schedules, projects, knowledge topics, sources, and learning units are represented as connected objects that share a consistent execution and evaluation model.

The system focuses on making structure explicit. Information is not only stored, but organized in a way that allows evaluation, prioritization, and interaction between domains such as planning, learning, and knowledge development.

Current development focuses on integrating a Zettelkasten-style knowledge system.

The architecture aims to make it possible for automated processes and intelligent agents to operate on the full system state, using tasks, knowledge, schedules, and context as input.

## Tech Stack

.NET · ASP.NET Web API · Avalonia UI · PostgreSQL · EF Core · REST · SignalR · Wolverine · RabbitMQ
