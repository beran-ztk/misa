# MISA

MISA is a personal orchestration system for work, knowledge, and long-term decision making.

Instead of using separate tools for tasks, notes, planning, and knowledge management, MISA models everything as structured items inside a single system. Tasks, schedules, projects, knowledge topics, notes, sources and learning units are represented as connected objects that share a consistent execution and evaluation model.

This structure allows information from different domains to interact, making evaluation and prioritization possible across the entire system.

Current development focuses on integrating a Zettelkasten-style knowledge system.

The architecture aims to make it possible for automated processes and intelligent agents to operate on the full system state, using the system context as input.

## Tech Stack

.NET · ASP.NET Web API · Avalonia UI · PostgreSQL · EF Core · REST · SignalR · Wolverine · RabbitMQ
